# Image Samples

These sample images are used for manual validation of the future UIMD `Image`
element.

Source: scikit-image data repository
https://gitlab.com/scikit-image/data

Files:

- `astronaut.png` - RGB portrait/scene sample.
- `camera.png` - grayscale camera sample.
- `chelsea.png` - RGB animal/photo sample.
- `coffee.png` - RGB still-life sample with smooth color gradients.
- `coins.png` - grayscale object/contrast sample.

The files were downloaded from the repository's `master` branch raw endpoints.
